# Spec_Space3D

## Overview

- **Genre**: 3D Platformer  
- **Engine**: Unreal Engine 5.32  
- **Concept**: ゲームの「不具合」を乗りこなすアクションゲーム  
- **Demo**:https://www.youtube.com/watch?v=RPq9iKV3Iq4

## Game Design

### ゲームの仕様

基本操作は横方向移動 + ジャンプのシンプルなプラットフォーマー。  
ただしプレイヤーは、自機の以下のパラメータを操作できる：

- 当たり判定
- 重さ
- 物理抵抗

いわばゲームにおける“当たり前の挙動”そのものを調整できる。

本来であれば不利益を生む「不具合」だが、  
うまく活用すれば強力な挙動へと変貌する“裏技”となる。

それらを最大限に利用してステージを攻略する、  
**パズル要素を含んだアクションゲーム**である。

---

## Design Philosophy

ゲームの不具合や、それを利用したいわゆる「バグ技」は、  
プログラマー視点では望ましくない挙動である一方で、  
プレイヤー視点ではこの上なく愉快な仕様として好まれる傾向がある。

これは、ゲームが「面白い」と認識される根幹にある  
**“制御感”** を強く刺激するためと考えられる。

ゲームの仕様を上手く利用してタスクを達成できた時、
脳の報酬系が活動して、ゲームが「面白く」感じられる。
特に、**より少ないリソースで**、**よりいい結果を得られた**時には、
ゲーム体験の制御感が増し、面白さは格段に強く感じられる。

バグは本来「条件外」で目標を達成するための手段であり、 
その制御感や支配感は、この「面白さ」を最大限に引き立てるものと考えられる。


**このゲームは、一見不利益で扱いづらい「不具合」のような仕様を利用し、
組み合わせることで、面白おかしくステージをクリアしていくゲームである。
